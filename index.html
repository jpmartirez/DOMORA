This System is designed to help users find the best apartments based on their preferences for proximity to key amenities such as schools, eateries, laundromats, and stores. It uses Dijkstra's Algorithm to calculate the shortest distances between apartments and amenities from a predefined graph. The system is built using HTML, CSS (TailwindCSS), and JavaScript for the front end and logic, with a focus on user interactivity and simplicity. This project is currently a prototype and serves as a proof of concept. It requires further development for scalability, real-time data integration (e.g., Google Maps API), and a more user-friendly interface to meet practical use-case standards.
