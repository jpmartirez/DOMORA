DESCRIPTION

This system is designed to create an interactive Google Maps application that visually represents various essential locations in a real-life area scenario (Location: Site Lucena City), such as schools, eateries, laundry shops, stores, and apartments. Its purpose is to help users, particularly students or residents, easily find and explore nearby amenities based on their geographical locations. The map integrates markers for each location category, represented by distinct icons, allowing users to click on markers to view detailed information such as the name and address of the location.

The images used in this system are custom markers that visually distinguish each category of locations:

  1. School Marker for educational institutions.
  2. Eatery Marker for dining places.
  3. Laundry Marker for laundry services.
  4. Store Marker for convenience and retail stores.
  5. Apartment Marker for residential apartments.
